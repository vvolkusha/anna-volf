Дз 1
1)git -система контроля версий,
а github - это сервис, предоставляющий
место для удаленных репозиториев
2)git можно использовать для управления версиями,
например при работе в команде, чтобы одни исправляли баги,
а другие придумывали фичи и не нарушали логики кода
друг друга
3)термины: локальный репрозиторий, децентрализированные СКВ
удаленный репрозиторий, clone, commit, pull, push
4) когда один что-то чинит, а другой улучшает, нужно сделать так,
чтобы они не мешали друг другу, и когда не нужно отслеживать
актуальность изменений
5) merge - все ветки сливаются в одну, сохраняя временной
порядок изменений
rebase - порядок изменений не сохраняется, одна ветка
подкрепляется к другой
6) в СКВ локальные копии между собой не взаимодейтсвуют,
только с сервером, а в децентрализованных локальные репрозитории
также взаимодейтсвуют друг с другом
